Каталог товаров Telegram Bot

Данный бот реализован на aiogram (Python) и предназначен для демонстрации каталога товаров с категориями, описаниями и фото. Оплата и заказ не реализованы — бот только показывает товары.

Возможности
- Главное меню с кнопкой "Категории"
- Просмотр товаров по категориям
- Описание и фото каждого товара
- Фото берётся из локального файла `Photos/Example.JPG` (можно заменить на свои)
- Без оплаты и заказов — только демонстрация

Установка
1. Клонируйте репозиторий или скопируйте файлы в папку.
2. Установите зависимости:
   ```sh
   pip install -r requirements.txt
   ```
   (или)
   ```sh
   pip install aiogram
   ```
3. В папке проекта создайте папку `Photos` и поместите туда файл `Example.JPG` (или свои изображения).
4. В файле `config.py` укажите ваш токен Telegram-бота:
   ```python
   BOT_TOKEN = "ваш_токен"
   LOG_LEVEL = "INFO"
   ```

Запуск
```sh
python bot.py
```

Запуск на Windows
1. Откройте папку проекта в проводнике.
2. Щёлкните правой кнопкой мыши по папке с зажатым Shift и выберите "Открыть окно PowerShell здесь" или "Открыть командную строку здесь".
3. Введите команду:
   ```powershell
   python bot.py
   ```
4. Если Python не установлен, скачайте и установите его с https://python.org.
5. Если команда не работает, убедитесь, что переменная среды PATH содержит путь к Python.

Структура
- `bot.py` — основной код бота
- `config.py` — настройки (токен)
- `Photos/Example.JPG` — фото для товаров

Как добавить свои товары и фото
1. В файле `bot.py` измените словарь `CATEGORIES`, укажите свои названия, описания и путь к фото.
2. Для каждого товара можно указать свой файл изображения в папке `Photos`.

Пример работы
- Пользователь запускает бота, выбирает категорию, видит список товаров.
- При выборе товара бот отправляет фото и описание.
